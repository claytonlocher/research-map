# research-map
Webmap for displaying the research of UIUC professors. 


## js libraries:
* [leaflet.js](http://leafletjs.com/)
* [Kartograph.js](http://kartograph.org/)
* [mapstraction](http://mapstraction.com/)
* [polymaps](http://polymaps.org/)

## Other possibly helpful resources
* [leaflet sidebar plugin](https://github.com/turbo87/leaflet-sidebar/)
* [GEOG 585: Open Web Mapping](https://www.e-education.psu.edu/geog585/)
* [GEOG 863 - GIS Mashups for Geospatial Professionals](https://www.e-education.psu.edu/geog863/node/1776)
